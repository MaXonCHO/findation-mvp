# Findation MVP - Кросс-брендовый подбор тональных средств

🎨 Найди свой идеальный тон с помощью AI!

## 🚀 Live Demo
[Открыть приложение](https://your-vercel-app.vercel.app)

## 📋 Описание
Findation - это веб-приложение для подбора идеальных тональных средств на основе:
- Твоих текущих продуктов
- Типа кожи и предпочтений
- Базы данных связей между оттенками

## ✨ Функциональность
- 🎯 **Выбор продуктов**: Бренд → Линейка → Оттенок
- 🧪 **Умный квиз**: Анализ типа кожи и предпочтений
- 🤖 **AI рекомендации**: Поиск эквивалентных оттенков
- 👝 **Косметичка**: Сохранение выбранных продуктов
- 🔗 **Система связей**: Граф эквивалентности оттенков

## 🛠️ Технологический стек
- **Frontend**: HTML5, CSS3, Tailwind CSS, Vanilla JavaScript
- **Backend**: Python, FastAPI, SQLite
- **База данных**: 3683 оттенка от GoldApple
- **API**: RESTful с документацией Swagger

## 📱 Как использовать
1. **Добавь свои тональные средства** через селекторы брендов и продуктов
2. **Пройди квиз** о типе кожи и предпочтениях
3. **Получи AI рекомендации** эквивалентных оттенков
4. **Сравнивай** и выбирай лучшие варианты

## 🚀 Развертывание

### Frontend (Vercel)
```bash
# Установи Vercel CLI
npm i -g vercel

# Разверни
vercel --prod
```

### Backend (Railway)
```bash
# Установи Railway CLI
npm install -g @railway/cli

# Войди в Railway
railway login

# Разверни
railway up
```
